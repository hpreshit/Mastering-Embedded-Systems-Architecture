Detailed Description

Integrate SPI, UART, LCD, LED functions run them concurrently in the mbed RTOS. The following four threads are implemented:

ECEN 5803 Mastering Embedded System Architecture Project 2 Module 3 Microcontroller Firmware

 1. Display the temperature on the LCD and to the PC
 2. Adjust the brightness of the RGB LED using a potentiometer
 3. Display an incrementing counter on the LCD and to the PC
 4. Blink an LED
